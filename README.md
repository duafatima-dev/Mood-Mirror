Mood-Mirror Website
Application of ICT project - A website that mirrors your mood and makes suggestions accordingly.
Mood Mirroring is a responsive website designed to help users track, visualize, and reflect on their emotional well-being. This platform provides a simple, intuitive interface for daily mood logging with visual feedback.
